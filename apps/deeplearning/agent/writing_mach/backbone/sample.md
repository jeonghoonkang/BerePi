
- 제목은 동형 암호화 기술 소개와 기업용 제조 데이터 시스템 응용

  - 1 챕터
    - 동형 암호화 개념
    - 기존 AES 등의 암호화에 비해, 기술적으로 차별화 되는 항목 
    - 정수형, 실수형 동형 암호화 성능 비교
    - 동형 암호화의 딥러닝 모델, LLM 적용 가능성 소개

  - 2 챕터
    - 딥러닝 모델의 보안을 위해 동형 암호화를 사용하는 것의 장점
    - 동형 암호화를 사용하지 않는 일반 암호화를 딥러닝 모델에 적용 가능한가
    - 동형 암호를 이용하면 기업들이 서로 모델을 전송, 교환 하지만, 내부 데이터를 뽑아서 저장할 수 없는 이유
    - 동형 암호 처리 속도를 빠르게 하기 위한 시스템 구현 방법

  - 3 챕터
    - 동형 암호 대표 기업 소개
    - 동형 암호 적용 프로젝트 리스트 생성
    - 동형 암호 서비스를 추진하기 위해 필요한 하드웨어 인프라 


- 작성방법
  - 병렬실행: true
  - 병렬 챕터: 4
  - 챕터 에이전트는 각 챕터 내용 작성, 4개 agent
  - 분량을 맞춰서 작성
  - main writer agent 는 1, 2, 3 챕터 작성중인 내용을 확인하고, 작성 방향이 변화하는 부분에 대해서, 다시 작성 방향을 맞추도록 지시
