Nextcloud WebDAV Cron Builder Static HTML

점검, 삭제, 이동, 분할 명령을 한 번에 생성

`apps/nextcloud/filesystem/checkup.py`, `cleanup.py`를 중심으로 입력값을 조합해 실행 명령과 `crontab` 라인을 바로 만들어 주는 정적 HTML입니다. 삭제는 기존 Python 스크립트를 사용하고, 이동과 분할은 일반 셸 명령을 생성합니다.

작업 선택

작업 유형에 따라 필요한 입력 필드와 생성되는 명령이 바뀝니다.

일정 설정

아래 다섯 칸이 그대로 crontab 앞부분으로 들어갑니다.

`crontab`은 연도 개념이 없으므로, 날짜 입력은 메모와 설명 생성에만 사용합니다.

명령 입력값

삭제는 설정 파일 기반, 이동과 분할은 경로 기반 명령을 생성합니다.

            삭제 실행 모드는 `cleanup.py --execute`가 원래 `confirm` 입력을 요구하므로,
